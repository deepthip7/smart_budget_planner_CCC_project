How DP Works
1
Build a table of size (items × budget) — each cell stores the best value achievable.
2
For each item, ask: take it or skip it? Pick the choice that gives more value.
3
Trace back through the table to find which items were actually selected.
4
Result: maximum value guaranteed within the exact budget — optimal every time.
Complexity
Time: O(N × W)
Space: O(N × W)
